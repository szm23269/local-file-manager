Features

すべての機能が無料

ローカルで完結するため、ファイルはどこにも送信されません。完全にプライベートに使えます。

🗂
デュアルペイン
左右2画面で同時にフォルダを操作。コピー・移動が直感的に行えます。タブにも対応。
🖼
サムネイル表示
画像・動画・PSD ファイルをサムネイルでプレビュー。スライダーでサイズも調整可能。
👁
プレビューパネル
テキスト・画像・動画・音声・PDF・CSV・ZIP など多様な形式をインラインプレビュー。
🌳
フォルダツリー
サイドパネルにフォルダツリーを表示。お気に入り登録でよく使うフォルダへ素早くアクセス。
🔍
高速ファイル検索
フォルダ内を再帰的に検索。大文字・小文字を区別しない柔軟な検索に対応。
✏️
バッチリネーマー
複数ファイルを一括リネーム。連番、接頭辞/接尾辞追加、大文字変換など多彩なモード。
💾
設定の自動保存
ウィンドウ位置・表示モード・タブなど、前回終了時の状態を自動復元します。
🎨
PSD サポート
Photoshop PSD/PSB ファイルの埋め込みサムネイルを表示。ファイルを開かずに確認できます。
🌐
ブラウザで動作
ネイティブアプリ不要。お使いのブラウザがそのまま UI になります。http://filie をブックマークするだけ。
🖱
ワンクリックインストール
ダブルクリックするだけで完了するGUI インストーラー付き。Python も含めすべて自動セットアップ。
Install

インストール方法

Python も含めすべて自動でセットアップします。インストール後は http://filie をブラウザのお気に入りに登録してください。

  • 1
    ZIP をダウンロードして解凍
    ダウンロードした ZIP をダブルクリックで解凍します。
    ⬇ macOS 版をダウンロード
  • 2
    「Filie Installer」をダブルクリック
    解凍フォルダ内の Filie Installer.app をダブルクリックします。インストーラーが起動し、ダイアログに沿って進めるだけで完了します。Python も自動でインストールされます。
  • 3
    「Filie を起動」をクリック & ブックマーク
    完了ダイアログの「Filie を起動」をクリックするとブラウザが開きます。以下の URL をお気に入りに登録してください。
    http://filie
⚠️ 「開発元が未確認」の警告が出た場合:
Filie Installer.app を右クリック →「開く」→「開く」をクリックしてください。

Uninstall

アンインストール方法

アンインストーラーを実行するだけで、すべての設定が自動的に削除されます。

  • 1
    アンインストーラーを実行
    ダウンロードした ZIP を解凍したフォルダ内でターミナルを開き、以下を実行します。
    chmod +x uninstall-mac.sh ./uninstall-mac.sh
  • 2
    確認して続行
    削除される内容が表示されます。y を入力して Enter を押すと削除が始まります。管理者パスワードの入力が必要です。
以下のものがすべて削除されます：
・アプリ本体 (~/.local/share/filie)
・起動スクリプト (~/.local/bin/filie)
・デスクトップの Filie.command
・ポートリダイレクト設定 (pfctl LaunchDaemon)
・/etc/hosts の filie エントリ

⚠️ Python 自体は削除されません。Python を削除する場合は別途対応してください。
  • 1
    アンインストーラーを実行
    ダウンロードした ZIP を解凍したフォルダ内の uninstall-windows.bat をダブルクリックします。管理者として実行されます。
  • 2
    確認して続行
    削除される内容が表示されます。y を入力して Enter を押すと削除が始まります。
以下のものがすべて削除されます：
・アプリ本体 (%LOCALAPPDATA%\Filie)
・デスクトップのショートカット
・ポートリダイレクト設定 (netsh portproxy)
・hosts ファイルの filie エントリ

⚠️ Python 自体は削除されません。削除する場合は「設定」→「アプリ」から行ってください。
Usage

使い方

キーボードショートカットを使うとより快適に操作できます。

🗂 基本操作

  • フォルダをダブルクリックで開く
  • ファイルをダブルクリックでプレビュー
  • 右クリックでコンテキストメニュー
  • ドラッグで複数選択（Shift / Ctrl+クリック）

⌨️ ショートカット

  • F5 コピー
  • F6 移動
  • Del 削除
  • F2 名前変更
  • F3 プレビュー表示切替
  • Ctrl+T ツリー表示切替
  • Ctrl+F 検索
  • V サムネイル/リスト切替
  • Tab 左右ペイン切替
  • Backspace 親フォルダへ

🖼 サムネイル表示

  • ツールバーの「サムネイル」ボタンで切替
  • スライダーでサイズを調整
  • 動画はホバーで再生開始
  • PSD ファイルも埋め込み画像を表示

🔖 タブとお気に入り

  • ペイン上部の「+」でタブを追加
  • タブをダブルクリックで名前変更
  • お気に入りパネルに頻出フォルダを登録
  • フォルダツリーで階層ナビゲーション

🔍 検索・リネーム

  • Ctrl+F でインライン検索バー表示
  • サブフォルダも再帰的に検索
  • Shift+F2 でバッチリネーマー起動
  • 連番・正規表現など多彩な置換モード

🛑 停止方法

  • macOS: .command のターミナルウィンドウを閉じる
  • Windows: タスクバーの Python プロセスを終了
  • ブラウザのタブを閉じても停止しません
FAQ

よくある質問

すべての処理はお使いの PC 上で完結します。ファイルやフォルダ情報が外部サーバーに送信されることは一切ありません。
ZIP に同梱のアンインストーラーを実行してください。

macOS: ./uninstall-mac.sh
Windows: uninstall-windows.bat をダブルクリック

詳細は アンインストール セクションをご覧ください。
インストーラーが Python の有無を確認し、なければ自動でインストールします。事前に準備する必要はありません。内部的には Python 3.12 以上を使用します。
デフォルトでは 127.0.0.1（ローカルホスト）のみにバインドされており、他の PC からはアクセスできません。セキュリティのためこの設定を推奨します。
ブラウザのアドレスバーに以下を入力してアクセスしてください。

http://filie

それでも開かない場合は: http://127.0.0.1:8000
インストーラーが OS のホストファイル (/etc/hosts) に 127.0.0.1 filie を自動登録します。これにより「filie」というホスト名がお使いの PC 上のアドレスに解決されます。また、ポート 80 → 8000 のリダイレクトも設定されるため、ポート番号なしでアクセスできます。すべてお使いの PC 内だけで完結する設定です。
Windows SmartScreen の警告です。署名のない .bat ファイルには自動的に表示されます。

対処: 「詳細情報」をクリック → 下部に表示される「実行」ボタンをクリックしてください。

Filie のコードはすべて GitHub 上で公開されており、何も隠れた処理はありません。
デスクトップに作成されたショートカットをダブルクリックしてください。

macOS: デスクトップの Filie.command をダブルクリック
Windows: デスクトップの Filie ショートカットをダブルクリック

ブラウザが起動して http://filie が自動で開きます。お気に入りに登録しておくと便利です。
macOS: デスクトップの Filie.command を開いたターミナルウィンドウを閉じてください。

Windows: タスクバーを右クリック →「タスク マネージャー」→ Python プロセスを終了してください。

ブラウザのタブを閉じるだけではサーバーは停止しません。
